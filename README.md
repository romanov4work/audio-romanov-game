# Озвучиваем Мультики 🎬🎤

Обучающая веб-игра для детей 6-11 лет для развития дикции и речевых навыков.

## Описание проекта

Игра создана для конкурса Учи.ру (22 апреля - 4 мая 2026). Помогает детям развивать:
- Дикцию и артикуляцию
- Эмоциональную выразительность речи
- Навыки озвучивания
- Творческие способности

## Особенности

- 🎮 Игровые сессии с разными заданиями
- 🎭 Озвучка мультиков с разными эмоциями
- 🗣️ Скороговорки и сложные слова
- 🎯 Распознавание речи через Whisper
- 🎨 UI в стиле Учи.ру
- 👶 Kids-friendly интерфейс

## Технологии

- **Фронтенд**: HTML5, CSS3, JavaScript (ES6+)
- **Распознавание речи**: Whisper API
- **Синтез речи**: ElevenLabs (детский голос)
- **Дизайн**: UI Kit Учи.ру

## Быстрый старт

### Вариант 1: Без установки (Web Speech API)
1. Откройте `index.html` в браузере Chrome/Edge
2. Разрешите доступ к микрофону
3. Играйте!

### Вариант 2: С Whisper (лучшее качество)
1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите Whisper сервер:
```bash
python whisper_server.py
```

3. В другом терминале запустите веб-сервер:
```bash
python -m http.server 8000
```

4. Откройте http://localhost:8000 в браузере

## Игровые режимы

1. **Скороговорки** (лёгкие и сложные) - тренировка дикции
2. **Сложные слова** - произношение трудных слов
3. **Озвучка мультиков** - эмоциональная выразительность
4. **Скорочтение** - развитие беглости речи
5. **Звук Р** - постановка звука Р
6. **Дыхательные упражнения** - правильное дыхание
7. **Артикуляционная гимнастика** - разминка речевого аппарата
8. **Мастер эмоций** - продвинутая озвучка
9. **Рассказчик историй** - озвучка целых сцен
10. **Эксперт скороговорок** - самые сложные задания
