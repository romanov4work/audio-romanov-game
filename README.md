# Озвучиваем Мультики 🎬🎤

Обучающая веб-игра для детей 6-11 лет для развития дикции и речевых навыков.

## Описание проекта

Игра создана для конкурса Учи.ру (22 апреля - 4 мая 2026). Помогает детям развивать:
- Дикцию и артикуляцию
- Эмоциональную выразительность речи
- Навыки озвучивания
- Творческие способности

## Особенности

- 🎮 Игровые сессии с разными заданиями
- 🎭 Озвучка мультиков с разными эмоциями
- 🗣️ Скороговорки и сложные слова
- 🎯 Распознавание речи через Whisper
- 🎨 UI в стиле Учи.ру
- 👶 Kids-friendly интерфейс

## Технологии

- **Фронтенд**: HTML5, CSS3, JavaScript (ES6+)
- **Распознавание речи**: Whisper API
- **Синтез речи**: ElevenLabs (детский голос)
- **Дизайн**: UI Kit Учи.ру

## Быстрый старт

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd PC_first_diction
```

2. Откройте `index.html` в браузере:
```bash
# Для Windows
start index.html

# Или используйте любой локальный сервер
python -m http.server 8000
# Затем откройте http://localhost:8000
```

3. Разрешите доступ к микрофону в браузере

## Игровые режимы

1. **Скороговорки** - тренировка дикции
2. **Сложные слова** - произношение трудных слов
3. **Озвучка мультиков** - эмоциональная выразительность
4. **Скорочтение** - развитие беглости речи

## Требования

- Современный браузер (Chrome, Firefox, Edge)
- Микрофон
- Интернет-соединение (для Whisper API)

## Разработка

Проект использует vanilla JavaScript без сборщиков для простоты запуска.
