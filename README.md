# Озвучиваем Мультики 🎬🎤

## 📋 Краткое описание

Полнофункциональная обучающая веб-игра для детей 6-11 лет, разработанная для конкурса Учи.ру (22 апреля - 4 мая 2026).

**Цель:** Развитие дикции, артикуляции и эмоциональной выразительности речи через игровые задания.

## ✨ Основные возможности

- 🎮 **4 игровых модуля** с 80+ заданиями
- 🎤 **Распознавание речи** через Whisper API (локально) или Web Speech API
- ⭐ **Система оценки** с звёздами за точность произношения
- 🔥 **Комбо-система** с множителями очков
- 🏆 **Достижения** за прогресс
- 💡 **Система подсказок** (3 подсказки на уровень)
- 📊 **Статистика игрока** с сохранением прогресса
- 🎨 **UI в стиле Учи.ру** с яркими анимациями
- 🔊 **Звуковые эффекты** и визуальная обратная связь
- 📱 **Адаптивный дизайн** для всех устройств
- 💾 **Автосохранение** прогресса в localStorage

## 🎯 Игровые модули

| Модуль | Описание | Заданий | Сложность |
|--------|----------|---------|-----------|
| 👅 Скороговорки и слова | Скороговорки + сложные слова для тренировки дикции | 23 | Лёгкая → Сложная |
| 🎭 Озвучка и дыхание | Эмоциональная озвучка + дыхательные упражнения + артикуляция | 22 | Средняя |
| ⚡ Скорочтение | Быстрое и чёткое чтение с ограничением по времени | 15 | Средняя |
| 🗣️ Как улучшить звуки | Практика сложных звуков (Р, Л, Ш, Ж, Ч, Щ, С) | 20 | Лёгкая |

**Всего: 80 заданий с прогрессивной сложностью**

## 🛠 Технологии

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Распознавание речи:** 
  - Whisper API (локальный сервер)
  - Web Speech API (fallback)
- **Backend:** Python + Flask (для Whisper)
- **Стиль:** UI Kit Учи.ру
- **Хранение:** localStorage

## 📊 Статистика проекта

- **Строк кода:** ~2500+
- **Файлов:** 15+

- **Заданий:** 80
- **Типов заданий:** 8 (скороговорки, слова, озвучка, дыхание, артикуляция, истории, скорочтение, звуки)
- **Коммитов:** 5+

## 🎮 Как играть

1. **Выберите модуль** из главного меню (4 варианта)
2. **Прочитайте задание** и подсказку
3. **Нажмите на микрофон** и произнесите текст
4. **Получите оценку** от 0 до 3 звёзд
5. **Собирайте комбо** за последовательные правильные ответы
6. **Открывайте достижения** и улучшайте свою дикцию!

## 🏆 Система оценки

- **⭐⭐⭐ (3 звезды)** - Отлично! 95%+ точность
- **⭐⭐ (2 звезды)** - Хорошо! 85%+ точность
- **⭐ (1 звезда)** - Неплохо! 70%+ точность
- **Попробуй ещё раз** - Меньше 70%

## 🔥 Бонусная система

- **Комбо-множитель:** x1.5 → x2 → x3 за последовательные правильные ответы
- **Бонус за скорость:** Дополнительные звёзды за быстрое выполнение
- **Достижения:** 7 уникальных достижений за прогресс
- **Подсказки:** 3 подсказки на каждый модуль

## 🐛 Troubleshooting

**Микрофон не работает?**
- Разрешите доступ в браузере
- Используйте HTTPS (для Web Speech API)
- Проверьте настройки приватности ОС

**Whisper не подключается?**
- Убедитесь, что сервер запущен на порту 9000
- Игра автоматически переключится на Web Speech API

**Звуки не воспроизводятся?**
- Проверьте настройки игры
- Требуется взаимодействие пользователя перед первым звуком
