# Размеры моделей Whisper

## Модели и их размеры:

| Модель | Размер | Скорость | Точность | Рекомендация |
|--------|--------|----------|----------|--------------|
| **tiny** | ~39 MB | Очень быстро | Базовая | Для тестов |
| **base** | ~74 MB | Быстро | Хорошая | **Рекомендуется** |
| **small** | ~244 MB | Средне | Отличная | Для продакшена |
| **medium** | ~769 MB | Медленно | Очень хорошая | Если есть место |
| **large** | ~1550 MB | Очень медленно | Максимальная | Требует GPU |

## Текущая установка:

**Используется модель:** `base` (74 MB)

**Общий размер установки:**
- Библиотека openai-whisper: ~5 MB
- Зависимости (torch, numpy и др.): ~500-800 MB
- Модель base: ~74 MB
- **ИТОГО: ~600-900 MB**

## Whisper уже локально на бэкенде! ✅

Ваш `whisper_server.py` - это и есть **локальный бэкенд**:
- Работает на `localhost:9000`
- Модель загружается в память при запуске
- Обрабатывает запросы локально, без интернета
- Данные не уходят в облако

## Где хранятся модели:

Windows: `C:\Users\Admin\.cache\whisper\`

Модель скачивается **один раз** при первом запуске сервера.

## Оптимизация размера:

Если нужно сэкономить место, можно использовать модель `tiny`:

```python
# В whisper_server.py измените:
model = whisper.load_model("tiny")  # вместо "base"
```

Это уменьшит размер с 74 MB до 39 MB, но немного снизит точность.

## Проверка размера:

```bash
# Размер установленных пакетов
pip show openai-whisper torch numpy

# Размер кэша моделей
dir %USERPROFILE%\.cache\whisper
```
