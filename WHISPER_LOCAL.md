# 🎤 Локальная установка Whisper

## Статус установки

✅ Python 3.14.4 установлен  
✅ pip 26.0.1 установлен  
⏳ Whisper устанавливается...

## Созданные скрипты

### 1. `install-whisper.bat`
Устанавливает Whisper и все зависимости локально.

**Запуск:**
```bash
install-whisper.bat
```

### 2. `start-whisper-server.bat`
Запускает локальный Whisper сервер на порту 9000.

**Запуск:**
```bash
start-whisper-server.bat
```

### 3. `start-game.bat`
Полный запуск игры:
- Проверяет, запущен ли Whisper сервер
- Запускает веб-сервер на порту 8000
- Открывает игру в Brave Browser

**Запуск:**
```bash
start-game.bat
```

## Как использовать

### Первый запуск (установка):
```bash
# 1. Установить Whisper (один раз)
install-whisper.bat

# 2. Запустить игру
start-game.bat
```

### Последующие запуски:
```bash
# Просто запустить игру
start-game.bat
```

## Что устанавливается

- `openai-whisper` - модель распознавания речи
- `flask` - веб-сервер для API
- `flask-cors` - поддержка CORS

## Модели Whisper

По умолчанию используется модель `base`:
- Размер: ~140 MB
- Скорость: быстрая
- Точность: хорошая для русского языка

Другие модели (можно изменить в `whisper_server.py`):
- `tiny` - самая быстрая, меньше точность
- `small` - баланс скорости и точности
- `medium` - высокая точность, медленнее
- `large` - максимальная точность, требует GPU

## Проверка работы

После установки проверьте:
```bash
# Запустить Whisper сервер
start-whisper-server.bat

# В другом терминале проверить
curl http://localhost:9000/health
```

Должен вернуть: `{"status": "ok", "model": "base"}`

## Troubleshooting

**Ошибка: Python не найден**
- Установите Python с https://www.python.org/downloads/
- Убедитесь, что Python добавлен в PATH

**Ошибка: pip не найден**
- Переустановите Python с опцией "Add to PATH"

**Whisper сервер не запускается**
- Проверьте, что порт 9000 свободен
- Проверьте установку: `pip list | grep whisper`

**Игра не распознаёт речь**
- Убедитесь, что Whisper сервер запущен
- Проверьте http://localhost:9000/health
- Разрешите доступ к микрофону в браузере
